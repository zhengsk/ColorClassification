# 颜色分类系统

这是一个基于 React 和 TypeScript 的颜色分类系统，使用 LAB 色彩空间进行颜色匹配。

## 功能特点

- 🎨 颜色选择和实时匹配
- 📊 基于 LAB 色彩空间的精确颜色匹配算法
- 🎯 可自定义的颜色调色板
- 📥 支持导入/导出调色板配置
- 🖱️ 直观的用户界面，支持拖拽和点击操作

## 技术栈

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide React (图标库)

## 开始使用

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 使用说明

1. **颜色选择**：使用颜色选择器选择一个颜色，或直接输入十六进制颜色代码
2. **查看匹配结果**：系统会自动显示最接近的预定义颜色类别
3. **管理调色板**：
   - 点击颜色方块可直接修改颜色
   - 点击 ✕ 按钮删除颜色（悬停时显示）
   - 点击虚线方块添加新颜色到该类别
   - 点击编辑图标可重命名类别
   - 点击删除图标可删除整个类别
4. **导入/导出配置**：
   - 点击"导出"按钮可复制配置或下载为 JSON 文件
   - 点击"导入"按钮可从文件或粘贴 JSON 恢复配置

## 颜色匹配算法

系统使用 LAB 色彩空间计算颜色之间的差异（Delta E），这是一种更接近人眼感知的颜色差异计算方法，比传统的 RGB 空间更准确。
